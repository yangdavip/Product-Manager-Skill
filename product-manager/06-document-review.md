# 文档自评审模块

> 产品经理核心能力之六：文档自评审与质量保证

---

## 适用场景

- 文档编写完成后的自我检查
- 评审前的预审
- 文档质量把关

---

## 自评审流程

```
文档编写完成 → 自评审检查 → 发现问题 → 二次修改 → 最终版本
```

---

## 评审结论

| 结论 | 说明 |
|------|------|
| 通过 | 文档质量满足要求，可进入下一阶段 |
| 修改后通过 | 存在少量问题，修改后可直接进入下一阶段 |
| 不通过 | 存在严重问题，需要重新编写 |
