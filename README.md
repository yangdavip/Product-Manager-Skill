# Product Manager Skill

专业产品经理核心能力Skill，适用于AI编程助手（如Claude Code、Cursor等）

## 包含能力模块

### 模块一：需求分析
- 用户分析框架
- 需求拆解方法
- 优先级评估矩阵
- 需求规格模板

### 模块二：架构设计
- 功能结构图
- 业务流程图
- 状态流转图
- ER图设计
- 架构分层

### 模块三：需求文档编写
- PRD标准结构
- SRS标准结构
- 功能需求详情模板
- 编写规范

### 模块四：原型设计
- 页面清单管理
- 页面注解模板
- 原型元素规范
- 交互规范

### 模块五：信息架构梳理
- 分析步骤
- 输出格式（XMind/Mermaid）
- 脑图结构模板

## 使用方式

### 方式1：加载Skill
在支持Skill的AI助手中，使用skill工具加载：
```
/product-manager
```

### 方式2：直接引用
在对话中引用SKILL.md文件内容

### 方式3：复制到项目
将SKILL.md复制到项目的`.agents/skills/`目录下

## 适用场景

- 新项目/新功能的需求调研
- PRD/SRS文档编写
- 系统架构设计
- 原型设计与交互规范
- 信息架构梳理与可视化
- 需求评审与方案讨论

## 工作流程

```
需求分析 → 架构设计 → 原型设计 → 文档编写 → 信息架构
```
